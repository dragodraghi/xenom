Engine Test

Atleta

Modalità:

Format & Standard

For Time, cap 15'. Row prima, run dopo, no pausa tra le due. Score = tempo totale. T.B. = dopo il row.

Standard: row distance lock al display, atleta non lascia il rower prima del completamento. Run all'aperto secondo percorso indicato. DNF al cap → conta i metri totali completati.
