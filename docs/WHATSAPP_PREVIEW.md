## Nota cache WhatsApp

WhatsApp puo tenere in cache la vecchia anteprima per alcune ore. Dopo il deploy, se l'immagine non cambia subito, condividi il link una prima volta con `?v=2` alla fine, ad esempio:

`https://benevolent-narwhal-1c8444.netlify.app/iscriviti?v=2`
